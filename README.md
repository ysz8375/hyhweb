# 编程猫后援会官网

## 安装

> 请确保电脑上安装node和npm

* 安装依赖组件
```
npm install
```

* 运行预览
```
npm run serve
```

* 打包dist文件夹
```
npm run build
```

* 检查文件
```
npm run lint
```
